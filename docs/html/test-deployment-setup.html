1. 개요 (Why this guide?)

TooTalk 정식 배포 = Apple Developer ID 인증서 ($99/년) + Windows EV code signing ($400+) 의무 retain. 테스트 단계 = 정식 인증서 부재. 본 매뉴얼 = 인증서 없이 임시 배포 path 명시 + 사용자 manual 안내 정합.

WindowsEV code signing certunsigned .exe + SmartScreen "More info" bypass

2.2 ad-hoc codesign 적용 (개발자)

TooTalk 빌드 = PyInstaller post-process 안에서 자동 adhoc sign retain. 검증: 
codesign -dv dist/TooTalk.app 2>&1 | head -5
# 기대 출력:
#   Identifier=com.toonation.tootalk
#   Format=app bundle with Mach-O thin (arm64)
#   CodeDirectory v=20400 ... flags=0x2(adhoc)
#   Signature=adhoc

adhoc 미적용 시점 수동 적용:

codesign --force --deep --sign - dist/TooTalk.app

2.5 사용자 install 절차 (최종 사용자 안내)

주의: adhoc 서명 binary = macOS Gatekeeper 차단 default. quarantine attribute 제거 의무.
  1. GitHub Release 페이지 안 zip 또는 DMG download
  2. DMG 열고 TooTalk.app 을 /Applications 폴더 안 drag
  3. Terminal 열고 quarantine 속성 제거: 
    xattr -rd com.apple.quarantine /Applications/TooTalk.app
  4. Finder 안 TooTalk.app double-click 실행
  5. 마이크 + 카메라 권한 요청 시점 "허용" click (Info.plist NSCameraUsageDescription + NSMicrophoneUsageDescription 정합)
대안 절차: Terminal 부재 사용자 = 우클릭 → 열기 → "그래도 열기" 2회 시도. quarantine 제거 시점 단일 클릭 진행. xattr 패스 권장.

4.2 PyInstaller hidden imports 검증 (개발자)

현재 issue (cycle 169.427 검증): dist/TooTalk.app 안 aiortc Python module bundle 부재. tootalk.spec 안 hiddenimports 누락. 통화 fire 시점 ImportError 차단.

fix:

# tootalk.spec
hiddenimports=[
    "aiortc", "aiortc.contrib.media",
    "aiortc.contrib.signaling",
    "av", "av.audio", "av.video",
    "cffi", "cryptography",
    "google.crc32c", "pylibsrtp",
]
binaries=collect_dynamic_libs("av") + collect_dynamic_libs("pylibsrtp")

4.3 macOS TCC 권한 grant

Info.plist 안 다음 entry 의무 retain:

<key>NSCameraUsageDescription</key>
<string>TooTalk 영상 통화 시점 카메라 사용</string>
<key>NSMicrophoneUsageDescription</key>
<string>TooTalk 음성 통화 시점 마이크 사용</string>

사용자 처음 실행 시점 마이크/카메라 권한 dialog 자동 표시. 거부 시점 시스템 설정 → 개인정보 보호 → 마이크/카메라 안 TooTalk 활성 의무.
